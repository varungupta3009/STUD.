# STUD.

A group project by the students of Semester III, Section 'C', PES University

## About

This is a mini class project for the course "Web Technologies - I (UE17CS204)" taught by Dr. Oshin Sharma, Assistant Professor, PES University

## Getting Started

To view or work with the Project, you will have to fulfil the recommended Prerequisites

### Prerequisites

The things you need to view the project (recommended software)

* [Google Chrome](https://www.google.com/chrome/) - To view the Website
* [WAMP](http://www.wampserver.com/en/) - Offline Web Server used to implement Server Side Functions
* [Atom](https://atom.io/) - To view and edit the Project Files
